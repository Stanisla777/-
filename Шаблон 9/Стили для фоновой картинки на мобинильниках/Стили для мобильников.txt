На мобильных телефонах не работает свойство background-attachment: fixed для экономии ресурсов оно отключено.

Но есть способ задавать его таким образом. 

.page-content {
  padding-top: 10px;
    padding-bottom: 30px;
    width: 100%;
    min-height: 100vh;
    overflow: hidden;
    position: relative;
}



.page-content:before{
    content: "";
    display: block;
    position: fixed;
    background: url("../img/bacground.jpg") center;
    left: 0;
    top: 0;
    width: 100%;
    height: 100%;
    z-index: -10;
    -webkit-background-size: cover;
    -moz-background-size: cover;
    -o-background-size: cover;
    background-size: cover;
}

page-content - это блок в котором находится вся страница, по аналогии с wrapper
