В этом примере разберем как можно задавать различную анимацию объектамю А так как обычно у нас анимация происходит при скроллинге в определенное место, то и рассмотрим пример, когда анимашку над объектом мы прописываем в таблице стилей, а время выполнения в скрипте.


Например у нас есть блок, который мы хотим анимировать появление, например чтобы раскачивался.
#blok16blok1{
    position: absolute;
    left: -8%;
    top: 221px;
    width: 350px;
    height: 175px;
    visibility: hidden;
}
Блоку мы задали с помощью visibility изначально невидимость.
Дальше, чтобы не подключать всю таблицу стилей анимашек мы находим там анимацию pulse и коируем всё, что сней связано. Таблица находится в папке "Анимирование объектов с помощью скачанной..." При этом мы создаём новый класс, который мы будем подключать в скрипте.
Всё что касается анимашки копировать всю её  не буду,она состоит из нескольких частей, скопирую лишь часть, чтобы показать, что мы создаем новый класс.
.pulse {
    visibility: visible!important;
    -webkit-animation-duration: 1s;
    animation-duration: 1s;
    -webkit-animation-fill-mode: both;
    animation-fill-mode: both;
}
В той таблице стилей мы копируем из стиля из animated в вкласс который создаем. Изначально например в классе .pulse будет написано
.pulse {
  -webkit-animation-name: pulse;
  animation-name: pulse;
}
Теперь переходим к скрипту

$(window).scroll(function(){
        if ( $(window).scrollTop() >= $($('#blok16zag')).offset().top-200 )  {


		$("#blok16blok1").addClass("pulse")

 };
    });
По-мойму всё понятно, доскролили до нужного момента и блоку #blok16blok1 добавили класс
