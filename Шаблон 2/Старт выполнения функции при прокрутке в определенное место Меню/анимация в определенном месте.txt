Фраза гласит, что если верхняя граница окна браузера(это задается $(window).scrollTop() ) достигнет значения, которое будет больше координат верхней границы блока #blok16zag, то происходит какое-либо действие. Проще говоря перейдет эту границу.


В этом примере я ставлю несколько условий для положения верхней границы окна. Здесь в меню, которое передвинается вместе со страничкой когда будем скролить на определенной секции, в меню будет появляться подчеркивание


$(window).scroll(function(){

        if ( $(window).scrollTop() <= $($('#neobyaz')).offset().top&& $(window).scrollTop() >= $($('#button1')).offset().top )  {
            $("#menu1").css({"border-bottom": "1px dashed"});
        }
        else if ($(window).scrollTop()<$($('#neobyaz')).offset().top){
            $("#menu1").css({"border-bottom": "none"});
        }
        else if ($(window).scrollTop()>$($('#vpodarok')).offset().top){
            $("#menu1").css({"border-bottom": "none"});
        }


});
Тоесть пишем, что когда верхняя граница окна выше координат верхней границы блока #neobyaz, но ниже границы блока #button1 появляется пунктир. При этом, когда страничка скролится вниз или вверх пунктир исчезнет. Меньше <= это значит окно странички находится выше какого-либо блока по отношению к началу 
<= это верхняя граница
>= это нижняя граница
